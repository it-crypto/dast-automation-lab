# DAST Automation Lab

This project demonstrates how to perform automated Dynamic Application Security Testing (DAST) using **OWASP ZAP** and **GitHub Actions**.

## What You Will Learn
- How DAST works
- Difference between baseline, full, and API scans
- How to run ZAP scans inside GitHub CI/CD
- How to analyze HTML vulnerability reports
- How to use public vulnerable applications (Juice Shop)

## Tools Used
- GitHub Actions (free)
- OWASP ZAP (open-source)
- OWASP Juice Shop (vulnerable app)
